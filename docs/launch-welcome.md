## 👋 Bienvenue dans la communauté Louis

Louis est une plateforme d'IA juridique **open-source, auto-hébergeable, souveraine**, portée par l'association **DataRing**. Sortie publique **v0.1.0 le {{ DATE }}**.

### Ce que vous pouvez faire ici

- 💬 **Général** — présenter votre cabinet, votre usage, votre setup
- 💡 **Idées** — proposer un cas d'usage métier, un workflow juridique réutilisable, un connecteur français qui vous manque
- 🆘 **Q&A** — poser une question sur l'installation, la configuration, l'usage. Avant : lire [`docs/`](https://github.com/DataRing/louis/tree/main/docs) et [`docs/faq.md`](https://github.com/DataRing/louis/blob/main/docs/faq.md)
- 📣 **Annonces** — release notes, ADRs, événements

### Ce qu'on attend de cette première semaine

1. **Retours d'installation** : ce qui a marché, ce qui a cassé, sur quel OS, à quelle étape
2. **Cas d'usage** : que feriez-vous avec Louis ? Quelle feature manque pour que vous l'adoptiez sérieusement ?
3. **Stress test** : on aimerait que des cabinets le déploient en interne et nous remontent ce qui ne fonctionne pas

### Code de conduite

Voir [`CODE_OF_CONDUCT.md`](https://github.com/DataRing/louis/blob/main/CODE_OF_CONDUCT.md). Soyez professionnels, respectueux, factuels — ce qui est attendu de toute communauté technique sérieuse.

### Signaler une vulnérabilité ?

**Pas en public.** Voir [`SECURITY.md`](https://github.com/DataRing/louis/blob/main/SECURITY.md) → email à `security@data-ring.net`.

### Roadmap

Voir [`README.md`](https://github.com/DataRing/louis/blob/main/README.md#roadmap-publique). Les sub-APIs PISTE étendues (Judilibre, JADE, INPI, BODACC), le rate-limit Redis, et la doc SecNumCloud arrivent en v0.1.x. L'i18n anglais et le project sharing en v0.2.

### Contribuer

Voir [`CONTRIBUTING.md`](https://github.com/DataRing/louis/blob/main/CONTRIBUTING.md). Les PR externes seront ouvertes formellement autour de **v0.2** (~Q4 2026). D'ici là : ⭐ + issues + discussions.
